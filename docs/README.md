# Bolt Ecosystem — Cross-Repo Documentation

Workspace-level design documents that span multiple repositories in the
Bolt Protocol ecosystem. Per-repo docs live in each repo's own `docs/`
folder; this directory is for decisions that affect boundaries between repos.

# Decisions (ADRs)

Dated decision records. One file per decision: `YYYY-MM-DD-slug.md`.

A decision file states: context, the decision, why, and what it supersedes (if
anything). Decisions are immutable once made — a change of mind is a NEW dated file
that names what it supersedes, never an edit.

Pre-OS decision memos (ADR-001-NATIVE-APP-ARCHITECTURE, BTR5_DECISION_MEMO, ...)
remain in `docs/` as frozen records; they were not moved to keep their history and
inbound references intact.
